# 👋 欢迎使用 Folio

Folio 是一个**纯本地、隐私优先**的 Markdown / HTML / 网页包(ZIP)阅读器,带轻量编辑。
不联网、不上传、不要账号——你的文件只在手机里。

---

## 📂 打开 & 新建

- 首页点 **「选择文件」**,从手机里挑 `.md` / `.html` / `.zip`。文件会复制一份进 App 本地库,原文件不动,之后在「最近」里随时再开。
- 点 **「＋ 新建空白文档」** 直接建一篇,命名后立刻开始写。

> 也可以在别的 App(微信 / 文件管理器)里点 `.md` 文件,选「用 Folio 打开」。

---

## ✍️ 编辑(Markdown)

- 阅读 `.md` 时,点右上角 **✏️** 进入编辑;再点一次回到预览。
- 上方有一条**格式工具栏**:加粗 `B`、斜体 `I`、标题 `H`、列表 `•` / `1.`、任务 `✓`、引用 `❝`、行内码 `</>`、代码块 `▣`、链接 `🔗`、分割线 `—`。
- 选中一段文字再点 `B`,会把它**包裹成加粗**;不选则插入符号、光标自动落到中间。
- 改完**自动保存**回本地库,可以再 **分享 / 导出**。
- 在**平板 / 横屏**上,编辑时是**左边源码、右边实时预览**;手机上是编辑与预览来回切换。

---

## 📖 阅读

- 点 **Aa**:切阅读主题(纸白 / 米黄 / 夜黑 / 渐变风景…)、字体、字号、代码配色。
- 点 **目录**:按标题快速跳转。阅读进度会自动记住,下次接着读。
- 正文里的图片:**点一下全屏**,双指捏合放大。

---

## 🏷️ 标签 & 收藏

- **长按**文件卡片 → 可以加 **标签**、设 **收藏**、重命名、删除。
- 主页的标签 **chip 条** 按标签筛选;末尾的 🗂 可以重命名 / 删除标签。
- 想清爽点:在 **设置 ⚙** 里可以关掉「卡片上显示标签」。

---

## 🌐 HTML / 网页包(ZIP)

- `.html` 直接渲染(出于安全,JavaScript **默认关闭**,需要时一键开启)。
- `.zip` 网页包解压后,会正确加载里面的相对资源(图片 / CSS)。

---

## 🔒 关于隐私

所有文件只存在你的手机本地,Folio **不收集、不上传任何东西**,也不需要联网。
